import { Aside, Tabs, TabItem } from '@astrojs/starlight/components';

These examples use RitsuLib Mod ID `MyMod`, which normalizes to `MY_MOD`. The game files always use flat JSON and category-qualified IDs. SpireLoc source files use short model names and may use any of the four equivalent formats below.

All source tabs share the same selection. Switch format in one table and the other tables, this page in Chinese, and the BaseLib comparison will follow.

The built-in action defaults to YAML. For the other tabs, pass `--source-format toml`, `--source-format nested-json`, or `--source-format flat-json` respectively.

<Aside type="note">
  Flat JSON can also be a source format. Its keys still use source-style IDs such as `ArcaneBolt`; `--model-id ritsulib MyMod` adds game prefixes later.
</Aside>

## `cards`

```json title="MyMod/localization/eng/cards.json"
{
  "MY_MOD_CARD_ARCANE_BOLT.title": "Arcane Bolt",
  "MY_MOD_CARD_ARCANE_BOLT.description": "Deal {Damage} damage.\nDraw 1 card."
}
```

<Tabs syncKey="localization-source-format">
<TabItem label="YAML">

```yaml title="Tools/localization/eng/cards.yaml"
ArcaneBolt:
  title: Arcane Bolt
  description: |-
    Deal {Damage} damage.
    Draw 1 card.
```

</TabItem>
<TabItem label="TOML">

```toml title="Tools/localization/eng/cards.toml"
[ArcaneBolt]
title = "Arcane Bolt"
description = """
Deal {Damage} damage.
Draw 1 card."""
```

</TabItem>
<TabItem label="Nested JSON">

```json title="Tools/localization/eng/cards.json"
{
  "ArcaneBolt": {
    "title": "Arcane Bolt",
    "description": "Deal {Damage} damage.\nDraw 1 card."
  }
}
```

</TabItem>
<TabItem label="Flat JSON">

```json title="Tools/localization/eng/cards.json"
{
  "ArcaneBolt.title": "Arcane Bolt",
  "ArcaneBolt.description": "Deal {Damage} damage.\nDraw 1 card."
}
```

</TabItem>
</Tabs>

## `characters`

The `banter.alive.endTurnPing` field shows that only the model ID segment changes; deeper key segments remain structured.

```json title="MyMod/localization/eng/characters.json"
{
  "MY_MOD_CHARACTER_SEER.title": "The Seer",
  "MY_MOD_CHARACTER_SEER.titleObject": "The Seer",
  "MY_MOD_CHARACTER_SEER.description": "A traveler guided by distant stars.",
  "MY_MOD_CHARACTER_SEER.banter.alive.endTurnPing": "The stars are aligned."
}
```

<Tabs syncKey="localization-source-format">
<TabItem label="YAML">

```yaml title="Tools/localization/eng/characters.yaml"
Seer:
  title: The Seer
  titleObject: The Seer
  description: A traveler guided by distant stars.
  banter:
    alive:
      endTurnPing: The stars are aligned.
```

</TabItem>
<TabItem label="TOML">

```toml title="Tools/localization/eng/characters.toml"
[Seer]
title = "The Seer"
titleObject = "The Seer"
description = "A traveler guided by distant stars."

[Seer.banter.alive]
endTurnPing = "The stars are aligned."
```

</TabItem>
<TabItem label="Nested JSON">

```json title="Tools/localization/eng/characters.json"
{
  "Seer": {
    "title": "The Seer",
    "titleObject": "The Seer",
    "description": "A traveler guided by distant stars.",
    "banter": {
      "alive": {
        "endTurnPing": "The stars are aligned."
      }
    }
  }
}
```

</TabItem>
<TabItem label="Flat JSON">

```json title="Tools/localization/eng/characters.json"
{
  "Seer.title": "The Seer",
  "Seer.titleObject": "The Seer",
  "Seer.description": "A traveler guided by distant stars.",
  "Seer.banter.alive.endTurnPing": "The stars are aligned."
}
```

</TabItem>
</Tabs>

## `ancients`

This table demonstrates both combinations requested by the game:

- Original ancient `DARV` with custom character `Seer`: `DARV` is preserved, while `Seer` receives the character category.
- Custom ancient `Mirror` with custom character `Seer`: both IDs receive their RitsuLib categories.

```json title="MyMod/localization/eng/ancients.json"
{
  "DARV.talk.MY_MOD_CHARACTER_SEER.0-0.ancient": "A new face in my collection!",
  "DARV.talk.MY_MOD_CHARACTER_SEER.0-0.next": "Reply",
  "DARV.talk.MY_MOD_CHARACTER_SEER.0-1.char": "I am only passing through.",
  "MY_MOD_ANCIENT_MIRROR.title": "Mirror",
  "MY_MOD_ANCIENT_MIRROR.epithet": "The Reflection",
  "MY_MOD_ANCIENT_MIRROR.talk.MY_MOD_CHARACTER_SEER.0-0.ancient": "I have seen your path already.",
  "MY_MOD_ANCIENT_MIRROR.talk.MY_MOD_CHARACTER_SEER.0-0.next": "Reply",
  "MY_MOD_ANCIENT_MIRROR.talk.MY_MOD_CHARACTER_SEER.0-1.char": "Then show me where it ends."
}
```

<Tabs syncKey="localization-source-format">
